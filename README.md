# Gourmet Spices - Site de Prezentare

Acesta este un site de prezentare pentru o companie fictivă de condimente premium, creat ca proiect demo.

## Descriere
Site-ul este construit folosind tehnologii web standard:
- **HTML5** semantic pentru structură.
- **CSS3** modern pentru stilizare (Flexbox, Grid, Animații).
- **JavaScript** pentru interactivitate (meniu mobil, validare forme).

## Structura Proiectului
- `index.html`: Pagina principală.
- `sofran.html`, `mustar.html`, `ghimbir.html`: Pagini de detaliu pentru condimente.
- `contact.html`: Pagina de contact cu formular funcțional.
- `style.css`: Fișierul de stiluri.
- `script.js`: Scripturile pentru funcționalitate.

## Cum să vizualizezi site-ul
Pentru a vedea site-ul live, vizitează link-ul GitHub Pages (dacă este activat) sau deschide fișierul `index.html` în browserul tău.
